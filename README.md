Grammarly
=========

Uberblick (German)
------------------

Grammarly ist ein KI-gestuetzter Schreibassistent, der Grammatik, Rechtschreibung, Zeichensetzung und Stil in Echtzeit prueft. Dieses Tool integriert sich direkt in Ihren Arbeitsablauf.

What This Tool Provides
------------------

Real-time grammar and spelling correction, tone detection, plagiarism checking, and vocabulary enhancement suggestions. Works across browsers, desktop applications, and mobile keyboards.

Systemanforderungen (German): Windows 10 oder neuer, 2 GB RAM, aktive Internetverbindung fuer die KI-Analyse.
